# Обучение устному переводу с использованием ИКТ

Данный проект является практической частью реферата на тему анализа эффективности использования информационно-поисковых систем (ИПС) и информационно-коммуникационных технологий (ИКТ) в процессе подготовки устных переводчиков.

## Цель работы
Продемонстрировать применение ИКТ в обучении устному переводу, а также оптимизацию поиска научной информации с помощью информационно-поисковых систем.

## Содержание сайта
Сайт включает следующие разделы:
- **Результаты проведённого исследования** 
- **Анализ эффективности использования каталогов и поисковых машин при научном поиске**
- **Методика поиска информации по теме исследования**

## Используемые технологии
- HTML, CSS, Bootstrap для фронтенда
- GitHub Pages для хостинга

## Ключевая концепция исследования
Интеграция ИПС и ИКТ в процесс обучения устному переводу повышает эффективность подготовки переводчиков за счёт оптимизации поиска научных источников, моделирования профессиональной деятельности и развития цифровых компетенций.
